# Steamdeck Fan Control

Script to install jupiter-fan-control on Ubuntu based distros. I tested it specifically on PopOS but it should work on most of them. Inspired by https://aur.archlinux.org/packages/jupiter-fan-control

## Download

To download it either use the download as zip option on the green "Code" button or
```
git clone https://github.com/nico359/jupiter-fan-control-ubuntu.git 
```

## Installation

Run the script with sudo like this: 
```
sudo ./steamdeckpop.sh
```

You may have to make it executable first with 
```
chmod +x steamdeckpop.sh
```
